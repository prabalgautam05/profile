NEXUS- THE DESKTOP AI ASSISTANT

The Nexus AI Chatbot is an intelligent assistant designed to perform a variety of tasks through natural language interaction. Built using Python and integrated with Google Generative AI, this chatbot can generate responses, execute system commands, and provide real-time information. The project features a user-friendly interface created with Tkinter, allowing users to interact with the chatbot via text input or voice commands.

Key Features:

  • Generative AI Integration: Utilizes Google Generative AI for generating intelligent responses.
  • Text-to-Speech: Converts text responses to speech using pyttsx3, with customizable voice options.
  • Voice Commands: Recognizes and executes voice commands using the speech_recognition library.
  • System Control: Can open applications, control system functions (shutdown, restart, log off, lock), and fetch system information (battery, CPU, RAM).
  • Web Search: Performs web searches and provides results.
  • Chat History: Saves and loads chat history, with options to export and clear the chat.
  • User Interface: A sleek and responsive UI built with Tkinter, featuring buttons for sending messages, loading/exporting chat history, and controlling speech.

This project demonstrates the integration of AI with practical applications, providing a seamless and interactive user experience.

Face Recognition Login System

The Facegate is a Django-based web application that allows users to register and log in using facial recognition technology. This project leverages the power of OpenCV and the face_recognition library to provide a secure and user-friendly authentication mechanism.

Features:

  • User Registration: Users can register by providing a username and capturing their face using their device's camera.
  • User Login: Users can log in by capturing their face, which is then compared to the stored face image for authentication.
  • Admin Interface: An admin interface is available for managing user data and images.
  • Static Files Management: Static files such as CSS, JavaScript, and images are managed and served efficiently.

Key Files:

  • manage.py: Django's command-line utility for administrative tasks.
  • settings.py: Configuration settings for the Django project.
  • urls.py: URL routing for the application.
  • views.py: Contains the logic for user registration, login, and welcome views.
  • models.py: Defines the UserImage model for storing user images.
  • requirements.txt: Lists the dependencies required for the project.
